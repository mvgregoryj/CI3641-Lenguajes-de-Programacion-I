El iterador misterio se apoya en el iterador ins para obtener todas las permutaciones de los elementos de un arreglo. Para este ejemplo es aprovechado el uso del iterador ins de manera que no se repita alguna permutación, para ello ins es llamado en una ocasión de esta manera: for i in ins(1, [2,3]) y también for i in ins(1, [3,2]), así para la primera retorna [1,2,3], [2,1,3], [2,3,1] y para la segunda retorna [1.3.2], [3,1,2], [3,2,1] siendo estas seis todas las permutaciones de [1,2,3].
